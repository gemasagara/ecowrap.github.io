# EcoWrap

A website for an eco-friendly packaging project made from natural, edible ingredients.

## Overview

EcoWrap is an innovative packaging solution designed to reduce plastic waste. This website showcases the project, its benefits, and the team behind it.

## Pages

- **Home** - About EcoWrap
- **Benefits** - Why EcoWrap matters
- **Tools and Ingredients** - What goes into making it
- **Benefits for Buyers** - Why consumers should care
- **Author Profiles** - Meet the team

## Technology

- HTML5
- CSS3
- JavaScript (Vanilla)
- Fully responsive design
- Deployed using Railway

## Project

Created for Indonesia International Applied Science Project Olimpiad (I2ASPO) 2025 by Team EcoWrap - MAN 13 Jakarta Selatan
